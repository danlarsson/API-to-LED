# API to GPIO

## Install
- mkdir Someting
- cd Someting
- git pull https://github.com/danlarsson/API-to-LED.git
- cd API-to-LED
- python -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt


## Run with
- source venv/bin/activate
- python main.py
